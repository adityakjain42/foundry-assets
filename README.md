# Foundry Browser Assets

Public browser-facing assets for the Foundry website.

The main game repository can stay private while the website loads cards, custom-card builder assets, and lore JSON from here.

Useful URL patterns:

```text
https://raw.githubusercontent.com/adityakjain42/foundry-assets/main/data/lore_archives.json
https://raw.githubusercontent.com/adityakjain42/foundry-assets/main/CardLibrary/L_ISHA_1.png
https://raw.githubusercontent.com/adityakjain42/foundry-assets/main/CardLibrary/EMPRESS_FA.png
https://raw.githubusercontent.com/adityakjain42/foundry-assets/main/CardLibrary/CardBuilder/Framev11.png
```

Large game packs and installers should stay in GitHub Releases, not this browser asset tree.
